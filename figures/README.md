\# Figures



This folder contains figures generated by the R scripts in `/scripts`.



The figures are organised according to the four main analysis steps in the project:



\- `01\_corpus\_overview`: overview figures based on the curated corpus metadata.

\- `02\_manual\_stance\_coding`: figures based on manual stance categories.

\- `03\_keyword\_analysis`: figures based on keyword and thematic term analysis.

\- `04\_sentida\_analysis`: figures based on Sentida sentiment analysis and comparison with manual stance coding.



Not all generated figures are included in the final report. The report uses selected figures from these folders, while the remaining figures are kept for transparency.



Unless otherwise stated, figures are based on `data/corpus\_metadata.csv`.
